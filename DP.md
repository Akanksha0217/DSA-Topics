# What is DP (Dynamic Programming)?

👉 DP means:
Solve big problem using solutions of smaller problems

🎯 Simple Definition
- 👉 Instead of solving same problem again and again ❌
- 👉 We store the answer and reuse it ✅


## 🔥 Why DP?
👉 Without DP:
- Same work repeated many times ❌

👉 With DP:
- Avoid repetition → faster solution ✅

### Example 
Fibonacci:
fib(5) = fib(4) + fib(3)

- Without DP: fib(3) calculated again and again ❌

- With DP: Store fib(3) → reuse

## 🧠 Two Types of DP
1️⃣ Top-Down (Recursion + Memory)
Solve → store → reuse later

2️⃣ Bottom-Up (Tabulation)
Start small → build big answer

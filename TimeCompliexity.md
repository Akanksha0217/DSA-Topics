# ✅ What is Time Complexity?

It tells you how fast your algorithm runs as input size increases.

👉 Not actual time (seconds)
👉 It measures growth of operations

## We represent time complexity using Big-O

### Common Types:

| Complexity | Example              | Meaning       |
| ---------- | -------------------- | ------------- |
| O(1)       | Access array element | Constant time |
| O(log n)   | Binary Search        | Very fast     |
| O(n)       | Loop                 | Linear growth |
| O(n log n) | Merge Sort           | Efficient     |
| O(n²)      | Nested loops         | Slow          |
| O(2ⁿ)      | Recursion (bad case) | Very slow     |
